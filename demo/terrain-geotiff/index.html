Vorbereitung:

  1. Führe im Projekt-Root pnpm build aus, damit ../../loader existiert.
  2. Führe bash ../../scripts/download-test-dems.sh aus, um Test-DEMs nach /tmp/v-map-test-dems/ zu laden.
  3. Starte einen lokalen Server, z. B. pnpm dlx http-server -p 4174 im Projekt-Root.
  4. Öffne http://localhost:4174/demo/terrain-geotiff/
